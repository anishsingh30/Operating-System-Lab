## 📌 Overview

This repository contains hands-on implementations of core **Operating System concepts** as part of the academic curriculum. Experiments cover process management, CPU scheduling algorithms, deadlock handling, memory management, and shell scripting — all built on **Ubuntu Linux** using **C** and **Bash**.

## 📚 Concepts Covered

- **Process Management** — Creating and managing processes using `fork()` system calls
- **CPU Scheduling** — FCFS, SJF, and Priority-based scheduling with turnaround & waiting time analysis
- **Deadlock Handling** — Banker's Algorithm for safe state detection and deadlock avoidance
- **Memory Management** — FIFO Page Replacement to understand page faults and frame allocation
- **Shell Scripting** — Writing Bash scripts with conditionals, loops, and Linux commands
